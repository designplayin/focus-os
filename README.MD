# FocusOS

Technology should optimize for intention, not attention.

A plain Chrome extension (Manifest V3) that transforms YouTube into a calm, search-first experience.

## Install

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder

No build step. No npm.

## Structure

```
manifest.json
content/
  content.js
  focus.css
popup/
  popup.html
  popup.css
  popup.js
assets/
  icon-16.png
  icon-48.png
  icon-128.png
```
